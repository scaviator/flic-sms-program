Personal Smart-Button SMS Alerts

Program description for toll-free messaging verification.

Purpose

This is a private, non-commercial alerting tool that sends a low volume of SMS notifications when a Flic smart button is pressed. Typical volume is a few messages per week.

Recipients (Closed List)

Messages are sent only to the program owner and a small set of family members/friends. Phone numbers are manually added and managed by the owner; there is no public signup.

Opt-In (Consent)

Each recipient has explicitly consented to receive these alerts. Consent is collected directly (verbal and confirmed via text), and numbers are added manually by the owner. No third-party lists are used.

Message Frequency

A few messages per week at most; no marketing or promotional content.

Sample Messages

  • Alert: The button was pressed. Please check in.
  • Test: This is a system check from the smart button.
  • Emergency: Assistance requested.

Opt-Out / Help

Recipients can reply STOP to stop all messages. Reply HELP for help or more information. These keywords are honored automatically by the messaging provider.
